# Read: 03 - HTML Lists, CSS Boxes, JS Control Flow

## HTML Lists

```
1. <ol> - ordered lists
2. <li> - each item
3. <ul> - unordered lists
4. <dl> - definition lists
5. <dt> - definition term
6. <dd> - definition
7. list can be nested
```

## CSS Boxes

```
1. box dimensions - width, height
2. limiting width - min-width, max-width
3. limiting height - min-height, max-height
4. Overflowing content - selector { overflow: <hidden or scroll>;}
5. border, padding - inside of box; margin - outside of box
6. border - border-width, border-style, border-color
7. padding - padding-top, padding-bottom, padding-left, padding-right
8. margin - margin-top, margin-bottom, margin-left, margin-right
```
